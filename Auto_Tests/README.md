# Автотесты для API
Это мои первые автоматические тесты, написанные на Python с использованием библиотеки `requests`.

## Содержание
*   **test_get_post.py** — проверяет GET и POST запросы к публичному API.
*   **Что проверяется:** статус-коды, наличие и корректность данных в ответе.

## Как запустить
1. Установи Python и библиотеку requests: `pip install requests`
2. Выполни в терминале: `py test_get_post.py`
