# Чек-лист Smoke-тестирования для SauceDemo

**Дата:** 14.07.2026
**Цель:** Быстрая проверка критической функциональности интернет-магазина после релиза.

---

##  Проверка 1: Доступность сайта
- [ ] Страница входа открывается (`https://www.saucedemo.com/`).
- [ ] Все элементы на странице отображаются (поля ввода, кнопка Login).

---

##  Проверка 2: Авторизация
- [ ] Авторизация с валидным логином/паролем (`standard_user` / `secret_sauce`).
- [ ] После входа открывается страница каталога `/inventory.html`.

---

##  Проверка 3: Отображение каталога
- [ ] На странице отображаются карточки товаров.
- [ ] У каждого товара есть название, цена и кнопка "Add to cart".

---

##  Проверка 4: Работа корзины
- [ ] При нажатии "Add to cart" у товара счетчик корзины (значок с цифрой) увеличивается.
- [ ] При нажатии на иконку корзины открывается страница `/cart.html`.

---

##  Проверка 5: Выход из системы (Logout)
- [ ] Нажатие на бургер-меню раскрывает пункт Logout.
- [ ] Нажатие на Logout возвращает на страницу входа.

---

## Результат проверки
- [ ] **ВСЕ ПРОВЕРКИ ПРОЙДЕНЫ** — Система стабильна, можно тестировать глубже.
- [ ] **ЕСТЬ ПРОБЛЕМЫ** — ошибка в названии товара.
