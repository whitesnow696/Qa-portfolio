# Отчет о негативном API-тестировании

**Цель:** Проверить, как API обрабатывает некорректные запросы.

---

## Тест N1: GET-запрос к несуществующему ресурсу

**Метод:** `GET`
**URL:** `https://jsonplaceholder.typicode.com/posts/999`

### Результат
*   **Статус-код:** `404 Not Found`
*   **Тело ответа:** `{}` (пустой JSON)

### Проверки
*   [x] Статус-код соответствует ожидаемому (404).
*   [x] Сервер вернул пустой ответ, а не ошибку 500 (это хорошо).
*   [x] Время ответа не превышает 500 мс.

**Вывод:** API корректно обрабатывает запрос к несуществующему ID — возвращает 404.
