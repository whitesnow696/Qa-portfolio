# Тест-кейс TC-001: Авторизация с валидными данными

## Информация
| Атрибут | Значение |
| :--- | :--- |
| **ID** | TC-001 |
| **Название** | Проверка успешной авторизации пользователя с валидными учетными данными |
| **Тип** | Позитивный / Smoke |
| **Приоритет** | High |
| **Ссылка на требование** | `REQ-LOGIN-001`

## Предусловия
1.  Открыта страница авторизации `https://www.saucedemo.com/`.
2.  Браузер: OperaGX 133.0.5932.39.

## Тестовые данные
| Поле | Значение |
| :--- | :--- |
| Логин (Username) | `standard_user` |
| Пароль (Password) | `secret_sauce` |

## Шаги выполнения
| Шаг | Действие | Ожидаемый результат |
| :--- | :--- | :--- |
| 1 | Ввести в поле **Username** значение `standard_user` | Текст отображается в поле ввода |
| 2 | Ввести в поле **Password** значение `secret_sauce` | Символы скрыты (отображаются как точки) |
| 3 | Нажать кнопку **Login** | Система обрабатывает запрос, страница перезагружается |

## Ожидаемый результат
1.  Пользователь успешно авторизован.
2.  Происходит редирект на главную страницу каталога товаров (`https://www.saucedemo.com/inventory.html`).
3.  В правом верхнем углу отображается кнопка **Open Menu** (бургер-меню).
4.  URL-адрес страницы изменился с `/` на `/inventory.html`.

## Постусловие
Нажать на бургер-меню → **Logout**, чтобы выйти из системы и оставить её в чистом состоянии.

## Результат
Проверка пройдена
