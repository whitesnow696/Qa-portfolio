# Баг-репорт BUG-002: Отсутствует подтверждение выхода при нажатии Logout

## Информация
| Атрибут | Значение |
| :--- | :--- |
| **ID** | BUG-002 |
| **Заголовок** | При выходе из системы отсутствует диалоговое окно подтверждения |
| **Компонент** | Web-приложение, функция выхода (Logout) |
| **Серьезность** | **Trivial** (Очень низкая, не влияет на данные) |
| **Приоритет** | **Low** (Исправлять в последнюю очередь) |
| **Окружение** | Windows 10, браузер Opera GX 132.0.5905.110 |

## Шаги воспроизведения (STR)
1. Авторизоваться на сайте `https://www.saucedemo.com/` (логин: `standard_user`, пароль: `secret_sauce`).
2. Нажать на иконку бургер-меню (три горизонтальные полоски в правом верхнем углу).
3. В выпадающем меню найти и нажать кнопку **Logout**.

## Фактический результат (FR)
Пользователь мгновенно разлогинивается и переходит на страницу входа (`https://www.saucedemo.com/`) **без какого-либо предупреждения или подтверждения**.

## Ожидаемый результат (ER)
При нажатии на **Logout** должно появляться модальное окно с вопросом: *"Вы уверены, что хотите выйти?"* и кнопками **"Да"** / **"Отмена"**.

## Дополнительная информация
- **Предполагаемая причина:** Функционал реализован "в лоб" без учета пользовательского опыта (UX).
- **Воспроизводимость:** Всегда (100%).
- **Влияние:** Пользователь может случайно выйти и потерять состояние корзины.

## Статус
Обнаружен
