# Отчет об API-тестировании: POST-запрос

**Цель:** Проверить, что метод POST создает новый ресурс.

**Метод:** `POST`
**URL:** `https://jsonplaceholder.typicode.com/posts`

### Результат
*   **Статус-код:** 201 Created (Успешно создан)

**Тело запроса (JSON):**
```json
{
  "userId": 1,
  "title": "Мой первый тестовый пост",
  "body": "Создан через Postman для портфолио",
  "id": 101
}
### Проверки
*   [x] Статус-код равен 201
*   [x] Все отправленные данные: `userId`, `title`, `body` вернулись без изменений
*   [x] В ответе появился новый ключ `id`
