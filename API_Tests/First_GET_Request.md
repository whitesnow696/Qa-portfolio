# Отчет об API-тестировании: GET-запрос

**Цель:** Проверить, что метод GET возвращает корректные данные существующего поста.

**Метод:** `GET`
**URL:** `https://jsonplaceholder.typicode.com/posts/1`

### Результат
*   **Статус-код:** `200 OK`
*   **Время ответа:** `804` мс

### Тело ответа (JSON)
```json
{
    "userId": 1,
    "id": 1,
    "title": "sunt aut facere...",
    "body": "quia et suscipit..."
}
### Проверки
*   [x] Статус-код равен 200 (Успех)
*   [x] Время ответа не превышает 1000 мс (Приемлемая скорость)
*   [x] В ответе присутствуют все ключи: `userId`, `id`, `title`, `body`
*   [x] Значение `id` в ответе совпадает с запрошенным (1)
*   [x] Поля `title` и `body` не пустые
