# Тест-кейс TC-002: Авторизация с невалидным паролем

## Информация
| Атрибут | Значение |
| :--- | :--- |
| **ID** | TC-002 |
| **Название** | Проверка отображения ошибки при вводе неверного пароля |
| **Тип** | Негативный |
| **Приоритет** | Medium |

## Предусловия
1.  Открыта страница авторизации `https://www.saucedemo.com/`.
2.  Браузер: OperaGX 133.0.5932.39

## Шаги выполнения
| Шаг | Действие | Ожидаемый результат |
| :--- | :--- | :--- |
| 1 | Ввести логин: `standard_user` | Поле заполнено |
| 2 | Ввести пароль: `wrong_password` | Поле заполнено |
| 3 | Нажать **Login** | **Страница НЕ перезагружается**, пользователь остается на странице входа |

## Ожидаемый результат
- Система отображает сообщение об ошибке: **"Epic sadface: Username and password do not match any user in this service"**.
- Сообщение выделено красной рамкой.
- Поле с паролем не очищается автоматически.

## Постусловие
Нажать кнопку очистки браузера или перезагрузить страницу для следующего теста.
