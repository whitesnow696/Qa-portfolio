# Баг-репорт BUG-001: Некорректное отображение названия товара в каталоге

## Информация
| Атрибут | Значение |
| :--- | :--- |
| **ID** | BUG-001 |
| **Заголовок** | В названии товара отображается служебная строка "Test.allTheThings()" |
| **Компонент** | Web-приложение, страница каталога товаров (`/inventory.html`) |
| **Серьезность** | **Minor** (Не влияет на функциональность, только на восприятие) |
| **Приоритет** | **Low** (Исправлять в последнюю очередь, после критических багов) |
| **Окружение** | Windows 10, браузер Opera GX 132.0.5905.110 |

## Шаги воспроизведения (STR)
1. Открыть браузер и перейти по адресу `https://www.saucedemo.com/`.
2. Ввести логин `standard_user` в поле **Username**.
3. Ввести пароль `secret_sauce` в поле **Password**.
4. Нажать кнопку **Login**.
5. На открывшейся странице каталога товаров найти карточку товара "T-Shirt (Red)".

## Фактический результат (FR)
В карточке товара отображается название:  
**"Test.allTheThings() T-Shirt (Red)"**

## Ожидаемый результат (ER)
В карточке товара должно отображаться название без служебной строки:  
**"T-Shirt (Red)"**

## Дополнительная информация
- **Предполагаемая причина:** Строка `Test.allTheThings()` является внутренним названием или маркером для разработчиков и, вероятно, осталась в коде по ошибке.
- **Воспроизводимость:** Всегда (100%).
- **Ссылка на требование:** Отсутствует в явном виде, но исходит из логики, что пользователь не должен видеть технические маркеры.
